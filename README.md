# 🎬 Moodie - Sistema de Recomendação de Filmes

O **Moodie** é o motor de um sistema inteligente de recomendação de filmes. Este projeto foi desenvolvido aplicando regras de negócio complexas (filtragem e cálculo matemático de *score*) validadas através de uma suíte robusta de testes unitários.

## Tecnologias Utilizadas
* **Java** (Linguagem principal)
* **JUnit 5** (Testes Unitários e Testes Parametrizados)
* **Mockito** (Criação de *Mocks* e *Stubs* para dependências externas)

## Como Executar os Testes no Eclipse
1. Clone o repositório.
2. Importe o projeto no Eclipse IDE como um projeto Java/Maven.
3. Navegue até a pasta `src/test/java`.
4. Clique com o botão direito no pacote `br.com.moodie` ou em uma classe de teste específica.
5. Selecione **Run As > JUnit Test**.
6. Acompanhe a aba do JUnit para visualizar a aprovação dos cenários (*Green Bar*).
